지도학습 
  - knn 알고리즘
    k-Nearest Neighbors
	 사용자가 지정해 주는 k개
	 - 새로운 데이터가 들어가면 거리가 가까운 k개 데이터를 확인 후, 예측을 수행.
	  - 회귀 - k개 데이터의 target 평균  
	  - 분류 - k개 데이터의 많이 가지고 있는 target의 예측.
	  
하이퍼 파라미터, 모델 파라미터
  하이퍼 파리미터 - 사용자가 조정할 수 있는 파라미터 (k, max_width...)
  모델 파라미터 - 모델이 결정해 준다. 선형회귀 y= w1*x1 + w2 * x2 + b
